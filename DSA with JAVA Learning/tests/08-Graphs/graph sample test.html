Question 2: (4 marks)
Write a Graph class in Java that implements an undirected graph, which contains the following members:

  • int [][] a; // An adjacency matrix  of size nxn.
  • int n; // Number of vertices. The vertices are indexed from  0  to  n-1.
  • char [] v; // The array characters, v[0]  is 'A', v[1] is 'B',...
  • static int INF = 99; // 99 is considered as infinity value. This variable is used for shortest path algorithm only.
  • void setDataFromFile(String filename) throws IOException - get adjacency matrix from file
  • void setDataFromArray(int [][] b) - set data from the array b by passing  b.length to n, and b[i][j] to a[i][j], i, j = 0,1,...,n-1 .
  • void visit(int i) - display the vertex  i  in  character format (i.e. it displays  'A' for  i=0, 'B' for i=1,...)  
  • void dispAdj() - display adjacency matrix.
  • void dispDegrees() - calculate and display degrees of all vertices.
  • void depthFirst(int k) - depth first traverse from the vertex  k.
  • boolean isConnected() - check and return true if a graph is connected, return false if it is not connected.
  • void breadthFirst(int k) - breadth first traverse from the vertex  k.
  • void dijkstra(int p, int q) - find and display the shortest path from the vertex  p  to the vertex  q  using Dijkstra algorithm.

Note: You can add other data members or methods to the class Graph, or even, you can create other classes if you see they are necessary, but if you are required, you should give a reason for that.
