Question 1: (6 marks)
Write a program in Java using linked list to manage information about students. Variables used to store information about a bus are:

  • code - the code of a student (character string).
  • name - the name of a student (character string).
  • age - the age of a student (integer value).
  • mark - the mark of a student (double value).

Conditions for valid data: The code does not contain 'B' character  and  mark must be between 0 and 10.
You should write the MyList class (and other necessary classes), which is a linked list data structure to store student information (The type of linked list is optional). The following functions should be incuded in the MyList class:

  • void clear() - clear the list.
  • void add(Student x) - add student object x  to the end of the list.
  • void add(String code1, String name1, int age1, double mark1) - check if data is valid then add new element to the end of the list.
  • void addMany(String [] a, String [] b, int [] c, double [] d, int n) - using the add function to add  students with  code  a[i], name b[i], age c[i] and mark d[i], i=0,1,..., n-1 to the end of the list (thus only students with valid data are added).
  • void display() - dislay all students in format:  (code, name, age, mark)
  • void deleLast() - delete the last node in the list.
  • void deleFirst() - delete the first node in the list.
  • Node search(String xCode) - search and return node whose code is equal to xCode
  • void dele(String xCode) - delete the first node in the list, whose code is xCode.
  • void dele(Node q) - delete the node q.
  • void sort() - Sort the list by code.
