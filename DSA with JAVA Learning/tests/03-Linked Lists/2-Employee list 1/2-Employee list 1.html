Question 1: (5 marks)
In this question you must write Java program using singly linked list to manage a list of  employees. Information about an employee is the following:

  • name - the name of an employee (character string).
  • salary - the salary of an employee (integer value).
  • bonus - the bonus of an employee (integer value).

Conditions for valid data: salary and bonus are positive and bonus<salary.
Beside the Main class, which is used to test your program, you should write 3 classes: Employee, Node and MyList. The form of these classes look something like below ( ... means other members):
class Employee {String name; int salary, bonus; ...}
class Node {Employee info; Node next;...}
class MyList {Node head,tail;...}
The following functions should be incuded in the MyList class:

  • void add(String name1, int salary1, int bonus1) - check if data is valid then add new element to the end of the list.
  • void addMany(String [] name, int [] salary, int [] bonus, int n) - using the add function to add  n  employees with  name  name[i] and salary[i], bonus[i], i=0,1,..., n-1 to the end of the list (thus only employees with valid data are added).
  • void displayAll() - dislay all employees in format:  (name, salary, bonus, income)
  • void deleLast() - delete the last employee in the list.

The Main.java to test your application is bellow:
public class Main

int [] salary = {800,200,900, 600};

h.displayAll(); // 2 marks
