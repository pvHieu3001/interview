Question 2: (5 marks)
In this question you must write Java program using binary search tree to manage employees. Information about an employee is the following:
Write a Java program to manage a list of employees. Information about a employee is the following:

  • code - the code of an employee (character string, which is the key of a tree).
  • name - the name of an employee (character string).
  • salary - the salary of an employee (integer value).
  • bonus - the bonus of an employee (integer value).

Conditions for valid data: salary and bonus are positive and bonus<salary.

Beside the Main class, which is used to test your program, you should write 3 classes: Employee, Node and BSTree. The form of these classes look something like below ( ... means other members):
class Employee {String code, name;  int salary, bonus; ...}
class Node {Employee info; Node left, right;...}
class BSTree {Node root;...}
The following functions should be included in the BSTree class:

  • void insert(String code1, String name1, int salary1, int bonus1) - check if data is valid then insert new element to the tree.
  • void insertMany(String code [], String [] name, int [] salary, int [] bonus, int n) - using the add function to add  n  employees with  code[i],  name[i], salary[i], and bonus[i], i=0,1,..., n-1 to the tree (thus only employees with valid data are inserted).
  • void visit(Node p) - display information in node p.
  • void preOrder(Node p) - pre-order traverse of the tree with root p.
  • void inOrder(Node p) - in-order traverse of the tree with root p.
  • void display() - display all node by breadth-first traverse.
  • void displayFirst(int k) - display first k elements in preorder traverse of the tree.

You should write the class Main to test the above functions.
