Question 1: (4 marks)
Write a program in Java using binary search tree to manage information about students. Variables used to store information about a bus are:

  • code - the code of a student (character string), which is the key of the tree.
  • name - the name of a student (character string).
  • age - the age of a student (integer value).
  • mark - the mark of a student (double value).

Conditions for valid data: The code does not contain 'B' character  and  mark must be between 0 and 10.
You should write the BSTree class (and other necessary classes), which is a binary search tree data structure to store student information. The following functions should be incuded in the BSTree class:

  • void clear() - clear the tree.
  • void insert(Student x) - check if  x is valid  then insert student object x  to the tree.
  • void insert(String code1, String name1, int age1, double mark1) - check if data is valid then insert new element to the tree.
  • void insertMany(String [] a, String [] b, int [] c, double [] d, int n) - using the insert function to insert  students with  code  a[i], name b[i], age c[i] and mark d[i], i=0,1,..., n-1 to the tree (thus only students with valid data are inserted).
  • void preOrder(...) - dislay all students in format:  (code, name, age, mark) by pre-order traversal. Parameters of the method are optional.
  • void inOrder(...) - dislay all students in format:  (code, name, age, mark) by in-order traversal.
  • void breadthTraverse() - dislay all students in format:  (code, name, age, mark) by breadth first traversal.
  • Node search(String xCode) - search and return a node, whose code is xCode the tree. Return null if such node not found.
  • int countNodes(Node p) - count and return number of nodes in a tree with root p.
  • int height(Node p) - return the height of a tree  with root p.
  • void copyDele(String xCode) - delete the node, whose code is xCode by copying.
