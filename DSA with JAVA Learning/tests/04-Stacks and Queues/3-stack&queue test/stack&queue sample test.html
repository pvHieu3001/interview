Question 1: (6 marks)
Write a MyStack class in Java that implements a stack ADT, whose elements  are String types.
The following functions should be incuded in the MyStack class:

  • boolean isEmpty() - return true if the stack is empty, i.e. the stack contains no elements, return false otherwise.
  • void clear() - clear the stack (after this method is called, the stack becomes empty).
  • void push(String x) - push a string x  into the stack.
  • String pop() - delete and return the top element from the stack (if the stack is empty then return null string).
  • void popAll() - pop and display poped element until the stack becomes empty.

Write a MyQueue class in Java that implements a queue ADT, whose elements  are integer types.
The following functions should be incuded in the MyQueue class:

  • boolean isEmpty() - return true if the queue is empty, i.e. the queue contains no elements, return false otherwise.
  • void clear() - clear the queue (after this method is called, the queue becomes empty).
  • void enqueue(int x) - enqueue value   x   into the queue.
  • int dequeue() - delete and return the front element from the queue (if the queue is empty then return 0 value).
  • void dequeueAll() - dequeue and display dequeued element until the queue becomes empty.
